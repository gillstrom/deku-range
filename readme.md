# deku-range

> Input range component for Deku


## Install

```
$ npm install --save deku-range
```


## Usage

```js
import Range from 'deku-range';

const render = () => <Range step={5} min={0} max={100} onChange={onChange} start={[10, 90]} connect/>;

export default {render};
```
